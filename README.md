# Multi-Modal AI System for Industrial Quality Assurance

SoC 2026 project - building AI systems for quality control using computer vision, ML, and LLMs.

## What's This About?

Learning to build AI systems that can analyze images, understand data, and use large language models for industrial quality assurance.

## Curriculum

**Week 1:** Python basics  
**Week 2:** Machine Learning fundamentals  
**Week 3:** Deep Learning & CNNs  
**Week 4:** Large Language Models  

## Tech Stack

Python, NumPy, Pandas, TensorFlow/PyTorch, OpenCV, Hugging Face

## Setup

```bash
git clone <repo-url>
cd multimodal-qa-system
pip install -r requirements.txt
```
